# ロリポップ！マネージドクラウド スターター for Ruby on Rails

## インストール

```ruby
gem install lolipop-mc-starter-rails
```

## Usage

Railsプロジェクトのソースコードがあるディレクトリで以下のコマンドを実行すると、ロリポップ！マネージドクラウドにデプロイするまでに必要な設定についてチェックしてくれます

```ruby
$ cd your_rails/
$ mc-rails check
```
